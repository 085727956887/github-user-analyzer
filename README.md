# GitHub User Analyzer

CLI sederhana untuk mengambil data profil GitHub menggunakan GitHub API.

## Fitur

- Cari profil GitHub
- Menampilkan nama
- Menampilkan followers
- Menampilkan repository publik
- Analisis level akun

## Teknologi

- Python
- Requests
- GitHub REST API

## Cara Menjalankan

```bash
pip install requests
python github_api.py
